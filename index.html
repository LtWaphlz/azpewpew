About this demo

This is a Norse/IPew‑style “pew‑pew” cyber attack map adapted for AZ DES leadership demos. Sources are sampled from global lists; with drill_mode=1 all arcs land at a chosen Arizona location (Phoenix, Tucson, Mesa). It’s an executive‑awareness visualization, not a forensic tool. Maintain context: colours/volume are illustrative, not real incident counts.

Controls: use the Zoom bar for city presets or mouse wheel + drag to explore. For a dramatic demo, toggle bad_day=1; to silence effects, add nofx=1. You can embed this page in a dashboard and swap destinations via URL parameters.

Code is from hrbrmstr/pewpew with light UI additions.
