About this demo

This is a Norse/IPew‑style “pew‑pew” cyber attack map adapted for AZ DES leadership demos. With drill_mode=1 all arcs land at a chosen Arizona location (Phoenix, Tucson, Mesa). It’s an executive‑awareness visualization, not a forensic tool. Colours/volume are illustrative.

Use URL params to change behavior (e.g., bad_day=1 for density, nofx=1 to mute, greenattacks/redattacks for palette). This page can be embedded in a dashboard.

Built on hrbrmstr/pewpew with light UI additions.
